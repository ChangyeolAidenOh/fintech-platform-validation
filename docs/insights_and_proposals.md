## 6. 프로젝트 한계

### 6.1 분석 단위의 차이

본 프로젝트는 상권×업종 단위의 공공 집계 데이터를 사용했습니다. 실제 카드사 비즈크레딧은 개별 사업자/가맹점 단위의 신용평가 및 휴·폐업 예측 모델입니다. 분석 단위가 다르므로, 본 프로젝트의 결과가 개별 가맹점 단위 모델을 직접 대체하거나 검증하는 것은 아닙니다. 외부 공공데이터의 증분효과를 검증하고, 내부 결제·가맹점 데이터 심화 분석이 더 높은 우선순위를 가질 수 있다는 방향성을 제시하는 것이 본 프로젝트의 범위입니다.

### 6.2 확보하지 못한 외부 데이터

인허가 정보, 사업자 등록 상태는 폐업의 직접적 법적 신호이므로, 본 분석의 환경 데이터와는 본질적으로 다른 성격의 외부 데이터입니다. 임대료 데이터(한국부동산원)도 해상도 제약(구 단위)으로 포함하지 못했습니다.

### 6.3 시간적 특수성

데이터 범위(2019Q1~2025Q3)에 COVID-19 충격과 이후 회복 국면이 포함되어 있으며, 이 시기의 소비 급감·회복 패턴이 모델의 일반화 성능을 왜곡할 수 있습니다.

### 6.4 공공데이터의 시간 해상도

분기 단위 집계로, 1~2개월 내 급격히 나타나는 폐업 징후를 포착하지 못했을 가능성이 있습니다.

### 6.5 Selection Bias

서울시 상권분석서비스 데이터로, 서울 지역 한정이며 골목상권·전통시장·관광특구 등 특정 유형에 집중되어 있을 수 있습니다.

### 6.6 Target 구조

62/63개 업종의 median closure_rate가 0이어서, median multiplier 기반 타겟은 사실상 "다음 분기 폐업 발생 여부"에 가까운 sparse-event classification으로 작동합니다. 이를 보완하기 위해 percentile 기반 타겟을 추가로 설계하여 검증했습니다.

## 7. 확장 방향

1. **개별 가맹점 단위 재검증:** 카드사 내부 데이터로 동일 ablation study를 수행하면, 공공 수준에서는 제한적이었던 외부 데이터가 개별 가맹점 수준에서는 다른 결과를 보일 수 있습니다.
2. **인허가/사업자 데이터 결합:** 법적 신호의 예측 기여도 검증.
3. **시계열 모델 확장:** LSTM/Transformer 기반 시계열 학습으로 매출 추이 자체를 시퀀스로 포착.

## 9. 결론

공공 외부데이터는 median 기반 sparse-event target뿐 아니라, 업종×분기 내 상대적 고위험군을 정의한 percentile target에서도 추정매출+점포 변수 대비 안정적인 평균 증분효과를 보이지 않았습니다. 이 결론은 5개 rolling time window, early stopping 방식 변경, 누수 검증에서도 일관되게 유지되었습니다.

정규화 타겟에서 외부 데이터의 SHAP 기여도가 30.5%까지 상승했으나, 이는 추정매출 피처와의 중복 신호(feature redundancy)이며 실제 예측력 개선으로 이어지지 않았습니다.

다만 매출·점포 정보가 모두 부족한 pure public-context 조건에서는 외부데이터만으로도 랜덤보다 높은 약한 선별력(AUROC 0.58~0.62)을 보였고, no-sales 조건에서는 percentile target 기준 AUROC +0.0053, AUPRC +0.0186이 개선되어, 공공 외부데이터는 일반적인 full-data 환경의 보강재라기보다 매출 정보가 부족한 상황의 약한 fallback signal로 해석됩니다.

본 프로젝트는 카드사 내부 데이터가 아닌 공공데이터 기반의 상권×업종 단위 검증이므로, 개별 가맹점 단위 모델을 직접 대체하는 것은 아닙니다. 공공 외부데이터의 증분효과를 다양한 타겟 정의와 검증 조건에서 비교함으로써, 대고객 금융 빅데이터 플랫폼에서 외부데이터를 "항상 붙이는 보강재"가 아니라 고객의 데이터 가용성에 따라 선택적으로 활용하는 context layer로 설계할 필요가 있음을 제안합니다.
