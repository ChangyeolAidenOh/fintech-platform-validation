## 0. 프로젝트 경위

BC카드 AI금융빅데이터플랫폼(ABP)에 직접 가입하여 AI검색·상권분석·데이터폴리오를 사용했다. ABP AI검색에 "카드 결제 데이터만으로 폐업을 예측할 수 있는가?"라고 질문했을 때, 플랫폼이 다음과 같이 응답했다:

> "카드 결제 데이터(추정매출)만으로 폐업을 완벽하게 예측하는 건 어렵습니다."
> "인허가 정보, 사업자 신고 상태, 주변 시장 상황 등 여러 데이터가 같이 참고되어야 정확도가 올라가요."

이 안내를 출발점으로, **ABP가 지목한 외부 데이터(유동인구, 상권변화지표, 집객시설, 거시경제지표, 상주인구)가 실제로 예측력을 개선하는지**를 서울시 공공데이터 290만 행으로 실증 검증했다.

## 1. 핵심 질문에 대한 답변

### Q1. 어떤 외부 데이터가 실제로 유효한가?

**답변: 공공 수준의 외부 데이터 중 유효한 것은 없다.**

7개 실험(순방향 ablation 5회, 역방향 ablation 1회, 전환점 예측 1회) 모두에서 외부 데이터의 증분 효과가 AUROC ±0.002 이내로 통계적 노이즈 범위였다. 역방향 ablation에서는 **외부 데이터를 전부 제거했을 때 오히려 AUROC가 +0.0018 개선**되어, 외부 데이터가 노이즈로 작용하고 있었음을 확인했다.

|---|---|---|
| 순방향 ablation (A→B) | 없음 | -0.0018 |
| + 유동인구 (C1) | 없음 | +0.0007 |
| + 상권변화지표 (C2) | 없음 | +0.0005 |
| + 집객시설 (C3) | 없음 | +0.0007 |
| + CSI (C4) | 없음 | -0.0016 |
| + 상주인구 (C5) | 없음 | +0.0010 |
| 역방향 ablation (전체 외부 제거) | 제거 시 개선 | +0.0018 |
| 괴리 피처 (카드×외부 교차) | 없음 | +0.0001 |

| store_count 통제 후 A vs B | 없음 | -0.0022 |

### Q2. 업종별로 유효한 신호가 다른가?

**답변: 업종별 리스크 드라이버는 다르지만, 차이의 원천은 외부 데이터가 아니라 카드+점포 데이터 내부에 있다.**

Kruskal-Wallis 검정에서 업종 간 폐업률 차이는 극도로 유의했다(H=36,494, p≈0). 그러나 업종별 SHAP 분석 결과, 상위 5개 드라이버는 거의 모든 업종에서 store_count, total_sales, total_txn_count 등 카드+점포 피처가 차지했다.

일부 업종(한식음식점, 커피-음료)에서 유동인구(total_foot_traffic)가 top 5에 진입했지만, 전체 기여도에서 차지하는 비중은 미미했다.

| 업종 | AUROC | 외부 피처 in Top 5 |
|---|---|---|
| 한식음식점 | 0.7534 | total_foot_traffic, change_index_numeric |
| 커피-음료 | 0.7769 | total_foot_traffic |
| 호프-간이주점 | 0.8007 | bus_stop_count |
| 미용실 | 0.7364 | 없음 |
| 일반의류 | 0.7892 | total_foot_traffic, total_facility_count |

### Q3. 외부 데이터를 결합하면 예측력이 얼마나 개선되는가?

**답변: 개선되지 않는다. 카드 결제 데이터 + 가맹점 정보만으로 AUROC 0.783, Precision@100 = 99%를 달성한다.**

상위 100개 리스크 상권 중 99개가 실제 폐업 — 실무적으로 즉시 사용 가능한 수준이다.

## 2. 주요 발견

### 발견 1: "카드 데이터만으로 안 된다"는 ABP의 안내는 부분적으로만 맞다.

ABP AI검색은 "외부 데이터를 결합해야 정확도가 올라간다"고 안내했다. 그러나 실증 결과, **공공 수준의 외부 데이터**(유동인구, 상권변화지표, 집객시설, CSI, 상주인구)는 카드 매출 + 가맹점 정보에 추가 가치를 주지 않았다.

다만 이것이 "모든 외부 데이터가 무의미하다"를 뜻하지는 않는다. ABP가 언급한 **인허가 정보**와 **사업자 신고 상태**는 본 분석에서 확보하지 못한 데이터이며, 이 데이터가 폐업의 직접적 법적 신호라는 점에서 공공 환경 데이터와는 성격이 다르다.

### 발견 2: 가맹점 리스크 예측에서 가장 강력한 신호는 "가맹점 자체의 구조적 특성"이다.

SHAP 기여도의 73%를 점포 컨텍스트(store_count, franchise_ratio, closure_rate, competition_density)가 차지했다. 이 변수들은 모두 **BC카드가 이미 보유한 가맹점 데이터**에서 파생된다.

| 피처 그룹 | SHAP 기여도 비중 |
|---|---|
| 점포 컨텍스트 (가맹점 DB) | 73% |
| 카드 매출 (결제 데이터) | 19% |
| 외부 데이터 전체 | 6% |
| 거시 경제 (CSI) | 2% |

### 발견 3: 상권변화지표는 직관과 반대로 작동한다.

서울시 상권변화지표에서 "다이나믹(LL)" 등급의 폐업률(0.046)이 "정체(HH)" 등급(0.038)보다 높았다. 통계적으로 유의하지만(chi-square p≈0), 방향이 기대와 반대다. "다이나믹"은 위험 신호가 아니라 개폐업 회전이 빠른 활성 상권을 의미한다.

이는 상권변화지표의 등급명이 소상공인에게 오해를 줄 수 있음을 시사한다.

### 발견 4: 76%의 상권-업종 조합에서 분기 내 폐업이 0건이다.

대부분의 상권-업종 조합은 안정적이며, 폐업은 특정 업종(치킨전문점 13.4%, 편의점 13.3%, 패스트푸드 10.5%)에 집중된다. 모든 업종에 동일한 리스크 모델을 적용하는 것보다, **고위험 업종에 특화된 모니터링 체계**가 효과적이다.

### 발견 5: 전환점(첫 폐업 발생) 예측에서도 외부 데이터는 유효하지 않다.

"이전까지 폐업이 없었던 상권에서 처음 폐업이 발생하는 시점"을 별도 타겟으로 설정하여 실험했다. 이 전환점에서도 외부 데이터의 기여는 없었다(AUROC -0.002). 환경 변화보다 가맹점 내부 구조가 전환을 결정한다.

## 3. BC카드 액션 제안

### 제안 1: 비즈크레딧 — 외부 공공데이터보다 자체 결제 패턴 심화에 투자

비즈크레딧 모델의 개선 방향은 유동인구/상권변화 같은 외부 공공데이터 확장이 아니라, BC카드가 독점적으로 보유한 **개별 가맹점 단위의 결제 패턴 심화 분석**에 있다.

본 분석에서 사용한 공공데이터는 상권 단위 분기 집계로, 개별 가맹점의 미시적 변화를 포착하지 못한다. BC카드의 실시간 트랜잭션 데이터에서 추출할 수 있는 다음 피처들이 더 유효할 가능성이 높다:

- 일별/주별 거래 빈도 변동 패턴 (감소 추세 감지)
- 객단가 변화 (동일 가맹점의 건당 결제액 추이)
- 결제 시간대 이동 (피크 시간 변화 = 고객층 변화 신호)
- 재방문 고객 비중 변화 (ABP에서 이미 제공 중인 지표)
- 동종 업종 내 상대적 매출 순위 변화

### 제안 2: ABP AI검색 — "외부 데이터가 필요하다"는 안내 문구 재검토

현재 ABP AI검색은 "카드 매출만으로는 폐업 예측이 어렵고 외부 데이터가 필요하다"고 안내한다. 이 안내는 사용자에게 ABP의 분석 능력을 과소평가하게 만들 수 있다.

실증 결과, **카드 매출 + 가맹점 구조 데이터만으로 Precision@100 = 99%의 리스크 스코어링이 가능**했다. ABP는 자사 데이터의 예측력에 더 자신감을 가져도 된다.

대안 안내 예시: "카드 결제 데이터를 기반으로 높은 정확도의 리스크 분석이 가능합니다. 인허가 정보, 사업자 등록 상태 등 법적 신호 데이터를 추가하면 정확도가 더 높아질 수 있습니다."

### 제안 3: CB사업 — 업종별 차등 모니터링 체계

폐업률이 업종별로 10배 이상 차이가 난다(치킨전문점 13.4% vs 일반의원 1.1%). 업종에 관계없이 동일 임계값을 적용하는 리스크 모니터링은 비효율적이다.

제안하는 3-tier 모니터링:

| Tier | 업종 예시 | 분기 폐업률 | 모니터링 주기 | 액션 |
|---|---|---|---|---|
| Tier 1 (고위험) | 치킨, 편의점, 패스트푸드, PC방 | 8~13% | 월간 | 선제적 리스크 경고 + 금융 지원 연결 |
| Tier 2 (중위험) | 커피-음료, 분식, 중식, 일식 | 5~7% | 분기 | 매출 추이 모니터링 |
| Tier 3 (저위험) | 의원, 치과, 가전, 컴퓨터 | 1~2% | 반기 | 정기 리포트 |

### 제안 4: ABP 상권분석 — 상권변화지표 활용 시 주의사항 표기

서울시 상권변화지표의 "다이나믹(LL)" 등급이 실제로는 폐업률이 가장 높다는 점을 ABP 상권분석 화면에 주석으로 표기하면 사용자 오해를 줄일 수 있다.

현재 상권분석 화면에서 상권변화지표는 별도 해석 없이 등급명만 제공되는데, "다이나믹 = 개폐업 회전이 빠른 상권으로, 신규 입점 기회와 폐업 리스크가 동시에 존재"라는 맥락을 추가하면 소상공인의 의사결정에 더 도움이 된다.

### 제안 5: ABP 데이터폴리오 — 예측 기능 추가 방향

현재 ABP는 현황 분석만 제공하고 예측은 제공하지 않는다. 본 분석의 모델이 보여주듯, **BC카드 자체 데이터만으로도 의미 있는 예측이 가능**하다. 데이터폴리오에 다음 예측 기능을 추가하면 상품 가치가 크게 높아진다:

- "이 상권-업종의 다음 분기 예상 폐업 리스크 등급" (High/Medium/Low)
- "이 상권-업종의 다음 분기 매출 성장 전망" (성장/유지/감소)
- "이 상권에서 가장 리스크가 높은 업종 top 3"

외부 데이터 결합보다 이 예측 기능 자체가 더 큰 상품 차별화 요소가 될 수 있다.

## 4. 프로젝트 한계 및 확장 방향

### 한계

1. **공공데이터 해상도의 한계.** 상권 단위 분기 집계 데이터로, 개별 가맹점 단위의 미시적 변화를 포착할 수 없다. BC카드의 실시간 개별 트랜잭션 데이터에서는 다른 결과가 나올 수 있다.

2. **확보하지 못한 외부 데이터.** ABP가 언급한 인허가 정보, 사업자 신고 상태, 임대료 데이터는 본 분석에서 확보하지 못했다. 특히 인허가/사업자 상태는 폐업의 직접적 법적 신호이므로, 이 데이터가 추가되면 결과가 달라질 가능성이 있다.

3. **타겟 변수 구성.** 상권×업종 단위의 폐업률을 사용했으나, 실제 비즈크레딧은 개별 사업자 단위로 운영된다. 분석 단위의 차이가 결과에 영향을 미칠 수 있다.

4. **시간 범위.** 2019~2025 데이터에 COVID-19 기간이 포함되어, 정상 시기와 비정상 시기의 패턴이 혼재되어 있다.

### 확장 방향

1. **BC카드 입사 후:** 개별 가맹점 단위 트랜잭션 데이터로 동일 분석 재수행. 공공 수준에서는 무효했던 외부 데이터가 개별 가맹점 수준에서는 유효할 수 있는지 검증.

2. **인허가/사업자 데이터 추가:** 공공데이터포털의 인허가 데이터, 국세청 사업자등록상태 API를 결합하여 "법적 신호"의 예측 기여도 검증.

3. **시계열 모델 확장:** 현재 분기별 스냅샷 피처를 사용했으나, LSTM/Transformer 기반 시계열 모델로 매출 추이 자체를 학습하면 예측력이 개선될 수 있다.

## 5. 결론 (한 문단)

BC카드 AI금융빅데이터플랫폼(ABP)은 "카드 매출 단독으로는 폐업 예측이 어렵고 외부 데이터가 필요하다"고 안내하고 있다. 서울시 공공데이터 290만 행, 7개 실험으로 이를 실증 검증한 결과, **공공 수준의 외부 환경 데이터(유동인구, 상권변화, 집객시설, CSI, 상주인구)는 카드 결제 + 가맹점 정보에 추가 예측력을 제공하지 않았다.** 카드 데이터만으로 Precision@100 = 99%의 리스크 스코어링이 가능했으며, 예측력의 73%는 BC카드가 이미 보유한 가맹점 구조 데이터에서 나왔다. 이는 BC카드의 데이터 사업 전략이 외부 공공데이터 확장보다 **자체 결제 데이터의 심화 분석과 예측 기능 상품화**에 더 큰 가치가 있음을 시사한다. 다만, ABP가 언급한 인허가 정보와 사업자 상태 데이터는 법적 신호로서 다른 성격의 외부 데이터이므로, 이에 대한 추가 검증이 필요하다.
